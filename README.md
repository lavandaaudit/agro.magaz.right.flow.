# Right Flow System - Конфігуратор системи

Сучасний інтерактивний конфігуратор для системи внесення рідких добрив **Right Flow**.

## Особливості
- **Інтерактивна смета**: Додавання компонентів, зміна кількості та миттєвий перерахунок.
- **Два типи рахунку**: Розрахунок з ПДВ (+20%) та без ПДВ.
- **Детальний опис**: Модальні вікна з технічними характеристиками кожного товару.
- **Адаптивний дизайн**: Оптимізовано для десктопів та мобільних пристроїв.
- **Преміальний стиль**: Темна тема з індустріальною естетикою та агро-логотипом.

## Технології
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Outfit)

## Встановлення
1. Клонуйте репозиторій.
2. Відкрийте `index.html` у вашому браузері.
